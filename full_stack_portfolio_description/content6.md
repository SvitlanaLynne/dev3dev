**Built with:** React

"Products" is a **front-end component** of a simple **online shop** built with React, created for practice purposes. It showcases fetching and displaying product data from an _API_ in a table format, _sorting_ and _filtering_ capabilities, _routing_ for navigation between pages and styling with the _Tailwind_ CSS framework.
